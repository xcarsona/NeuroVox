# Data Directory

This folder contains all datasets used for the Resonance AI project.

## Structure
- raw/        → Original unmodified datasets
- processed/ → Cleaned and feature-engineered data

## Data Sources
- DAIC-WOZ Depression Dataset (Primary)
- Kaggle Mental Health Datasets (Backup)

## Ethics
All datasets are:
- De-identified
- Approved for academic research
- Used only for non-clinical research

No raw data containing identifiable information will be publicly distributed.
